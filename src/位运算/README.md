位运算：

- `&`： 两个都是 1 的时候结果为 1

```
示例：判断奇数偶数
`num % 2` 效果等于 `num & 1`
```

- `|`： 两个任何一个为 1，结果为 1
- `^`： 两者相同结果为 0，两者不同结果为 1

```
leetcode 136

任何数字和自己亦或结果为0
任何数字和0抑或结果为自己
```

- `!`： 取反
- `>>` ： 相当于除以 2，然后取整
- `<<` ： 相当于乘以 2

# React

每个二进制都是乘以 2 的关系

add：0001
Update: 0010
replace: 0100
delete: 1000

如果某一步骤需要组合两种操作，将两种操作按位或一下。
update | replace


react执行更新的时候将最终的操作结果与每一种操作按位与一下
